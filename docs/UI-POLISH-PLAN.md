# 올A+ UI 디자인 다듬기 계획

> 작성일: 2026-04-01 | Phase 7 완료 후 UI 품질 향상

## 현재 상태
- shadcn v4 (Base UI) + Tailwind CSS + OKLCH 색상
- 다크모드 구현 완료 (next-themes)
- 모바일 바텀 내비게이션 구현 완료
- 기본적인 카드/버튼/다이얼로그 UI 존재
- **문제**: 전체적으로 "기본 shadcn" 느낌, 브랜딩 약함, 시각적 위계 부족

## 레퍼런스
- shadcn-admin (satnaing) — 접이식 사이드바, 글로벌 커맨드 검색
- Google NotebookLM — AI 채팅 + 소스 패널 + 생성 도구
- ChatGPT/Claude — 카드 기반 AI 응답, 마크다운 렌더링
- Quizlet — 플래시카드 UI, 진행률 표시
- Notion AI — 사이드바 구조, 인라인 AI

## Phase A: 핵심 비주얼 개선 (우선순위 높음)

### A1. 브랜딩 강화
- [ ] 로고 타이포 개선 — "올A+" 텍스트를 커스텀 로고로 (gradient 또는 아이콘 조합)
- [ ] 브랜드 컬러 체계화 — indigo-600 단일 → primary/accent/success 3색 체계
  - Primary: indigo-600 (메인 액션)
  - Accent: violet-500 (AI 관련)
  - Success: emerald-500 (학습 완료/긍정 피드백)
- [ ] 파비콘 + OG 이미지 추가

### A2. 대시보드 리디자인
- [ ] 과목 카드 개선 — 그라데이션 상단 바 + 통계 표시 (대화 수, 자료 수, 최근 활동)
- [ ] 학기 개요 섹션 — 상단에 요약 카드 (총 과목, 이번 주 학습 시간, 업로드 자료 등)
- [ ] 빈 상태 일러스트 개선 — 현재 아이콘만 → SVG 일러스트레이션

### A3. 채팅 UI 고급화
- [ ] AI 응답 카드 스타일 — 버블 대신 넓은 카드 (ChatGPT/Claude 스타일)
- [ ] 마크다운 렌더링 개선 — 코드 블록 하이라이팅 (rehype-highlight), 테이블, 수식
- [ ] 출처 표시 개선 — 인라인 [1][2] 번호 + 하단 참고자료 패널 (Perplexity 스타일)
- [ ] 타이핑 인디케이터 — 바운스 점 대신 "AI가 답변을 작성하고 있어요..." 텍스트 + 스켈레톤

## Phase B: 레이아웃 & 내비게이션 (중간 우선순위)

### B1. 사이드바 개선
- [ ] 접이식(collapsible) 사이드바 — 아이콘만 보이는 축소 모드
- [ ] 과목 색상 dot → 세로 바 또는 좌측 보더
- [ ] 검색 기능 — 사이드바 상단에 과목/대화 검색

### B2. 페이지 헤더 통일
- [ ] 브레드크럼 네비게이션 — "내 과목 > 데이터베이스 > 대화 #3"
- [ ] 페이지별 액션 버튼 영역 통일 (우측 상단)

### B3. 채팅 페이지 레이아웃
- [ ] 사이드 패널 — 대화 목록 + 소스 문서를 우측 패널로 (NotebookLM 참조)
- [ ] 입력창 하단 고정 + 최대 너비 제한 (max-w-2xl)

## Phase C: 마이크로 인터랙션 & 디테일 (낮은 우선순위)

### C1. 애니메이션
- [ ] 페이지 전환 — framer-motion으로 부드러운 전환
- [ ] 카드 호버 — scale + shadow 조합
- [ ] 메시지 등장 — slide-up + fade-in
- [ ] 스켈레톤 → 콘텐츠 전환 — 자연스러운 fade

### C2. 타이포그래피
- [ ] 헤딩 크기 체계화 — text-2xl(h1), text-xl(h2), text-lg(h3) 통일
- [ ] 본문 line-height — leading-relaxed 통일
- [ ] 코드 폰트 — Geist Mono 적용 확인

### C3. 다크모드 세밀 조정
- [ ] 하드코딩 컬러 제거 — bg-gray-50, bg-white, text-gray-600 등 → CSS 변수
- [ ] 그림자 조정 — 다크모드에서 shadow 대신 border 강조
- [ ] 이미지/아이콘 밝기 조정
